The Inner Workings of DootDoot

DootDoot uses a variety of technologies to be able to achieve the capabilities it has. The website uploadds the .wav file to a Flask server. This then preprocesses the audio file by reducing background noise and cropping out unnecessary silences, as well as implementing convolutional waveform smoothing to make changes in the audio more evident. The program then extracts various features such as the BPM. Using the BPM, troughs, and peaks found in the .wav file, the file is split into individual notes. These notes are converted into chromagrams and the pitch is analyzed to get a the note value. This value is then further validated by a custom deep Convolutional Neural Network model that was built from scratch. The note's waveform is converted into a spectrogram and fed into the deep CNN model to get the note value. Once we have all the notes, we sequence them into a sheet music pdf and send it back to the website for the user to download.
